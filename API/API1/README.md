# API1 – School Library REST API Experiment

## Beschrijving
Dit experiment maakt gebruik van de **School Library REST API**.  
Het doel is om basis **CRUD-operaties** uit te voeren met zowel **curl** als **Python**.

## Doelstellingen
- REST API gebruiken
- Werken met HTTP methods (GET, POST, DELETE)
- Authenticatie via API token
- curl en Python vergelijken

## Gebruikte tools
- curl
- Python 3
- requests library

## Oefeningen
- GET boekenlijst
- POST nieuw boek
- DELETE boek via ID

## Conclusie
Deze oefening toont aan dat ik REST API’s correct kan gebruiken met zowel curl als Python.

# API6 – REST API Experiment with curl (Forms)

## Beschrijving
Dit experiment demonstreert het gebruik van **curl met form-data** (`-F`) om POST requests te versturen.

## Doelstellingen
- POST requests met form-data
- multipart/form-data begrijpen
- API responses analyseren

## Gebruikte tools
- curl
- httpbin.org (test API)

## Conclusie
Deze oefening toont aan dat ik REST API’s kan testen met webforms via curl.

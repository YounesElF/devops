# API2 – REST API Integration with Python

## Beschrijving
In dit experiment wordt een REST API volledig geïntegreerd met **Python** zonder curl.

## Doelstellingen
- REST API aanspreken met Python
- requests.get(), requests.post(), requests.delete()
- JSON responses verwerken
- HTTP statuscodes controleren

## Gebruikte tools
- Python 3
- requests

## Conclusie
Deze oefening toont aan dat ik een REST API kan integreren in een Python applicatie.

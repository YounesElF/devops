# Docker Experiments – DEVASC

Deze map bevat mijn **Docker-oefeningen** uitgevoerd in het kader van **DevNet / DEVASC**.  
De oefeningen behandelen het gebruik van **Docker containers en images** voor applicatie-deployment.


## Wat wordt aangetoond
- Werken met Docker images en containers
- Containers starten, stoppen en beheren
- Poorten mappen tussen host en container
- Eigen Docker images bouwen
- Multi-container applicaties begrijpen

## Gebruikte commando’s
- `docker run`
- `docker ps`
- `docker stop`
- `docker rm`
- `docker build`

## Conclusie
Deze map toont mijn **praktische kennis van Docker** en containerisatie binnen **DEVASC**.

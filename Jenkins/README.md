# Jenkins – Oefeningen & Pipelines

## Doel
Deze map bevat **Jenkins-gerelateerde DevNet oefeningen**
gericht op CI/CD, Docker automatisatie en pipeline-denken.

## Structuur
- `j1_lab_6_3_6/`  
  Officieel DevNet lab (Lab 6.3.6)

- `j2_pipeline/`  
  Eigen pipeline-experiment (uitbreiding op J1)

## Jenkins vereisten
- Jenkins draait op Linux
- Docker is geïnstalleerd op dezelfde host
- Jenkins user heeft toegang tot Docker
  (meestal lid van `docker` groep)

## Belangrijke DevNet concepten
- CI/CD pipelines
- Infrastructure as Code
- Automatisatie van build, test en deployment
- Herhaalbaarheid en cleanup

## Opmerking
Deze oefeningen gebruiken **bewust eenvoudige applicaties**
zodat de focus ligt op:
➡️ pipeline-logica  
➡️ Docker-commands  
➡️ foutanalyse en debugging

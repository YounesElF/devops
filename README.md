# DevOps Portfolio – DEVASC

Dit repository bevat mijn **DevOps- en automatisatie-oefeningen** uitgevoerd in het kader van **DevNet / DEVASC**.  
De oefeningen behandelen verschillende DevOps-domeinen zoals **API’s, Flask, Docker, Ansible, Jenkins, unit testing en virtual environments**.

Het doel van dit portfolio is het aantonen van mijn **praktische DevOps-vaardigheden**.

---

## Overzicht per onderdeel

### API
- REST API gebruik met curl en Python  
- HTTP methods: GET, POST, DELETE  
- Authenticatie met tokens  
- JSON verwerking  

### Flask
- Microweb applicaties  
- Login en authentication services  
- REST microservices  

### Docker
- Containers starten en beheren  
- Eigen Docker images bouwen  
- Webservices en databases in containers  

### Ansible
- Playbooks en inventory files  
- Service deployment en configuratie  
- Automatisatie van Linux taken  

### Jenkins
- Basis CI/CD pipelines  
- Jobs en build automatisatie  

### Unittest
- Python unittest framework  
- Functies testen met assertions  
- Test data en foutafhandeling  

### Webex
- Webex REST API integratie  
- Bearer token authenticatie  
- Rooms beheren en berichten versturen  

### venv
- Virtual environments maken en gebruiken  
- Dependency management  
- Eenvoudige deployment experimenten
